**react-switch** 

- Switch parameters:
- get value
- Case parameters:
- appropriate dom
- Default parameters:
- another dom

## Usage

`    npm install react-switch`

`   import Switch,{Case,Default} from 'react-switch'`

## Example

```
<Switch value=''>
  <Case value=''>
    ...
  </Case>
  <Case value=''>
    ...
  </Case>
  <Default>
    ...
  </Default>
</Switch>
```
